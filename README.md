# WhatsApp Web Privacy Shield & Lock 🛡️

**Developed by Randun Labs**

A sophisticated Userscript for WhatsApp Web that adds a password lock screen and automatic privacy blurring.

## 🚀 Features
* **🔒 Lock Screen:** Blocks WhatsApp immediately upon loading. Requires a password to access chats.
* **👁️ Privacy Blur:** Automatically blurs messages, contact names, and images once unlocked.
* **🖱️ Hover Reveal:** Hover over any blurred element to reveal it temporarily.
* **⚙️ Granular Control:** Toggle specific blurs (Messages, Names, Images) via the menu.

## 📦 Installation
1.  Install [Tampermonkey](https://www.tampermonkey.net/) for your browser.
2.  [Click here to Install](https://greasyfork.org/en/scripts/567734-whatsapp-web-lock-privacy-shield-v7-0)

## 🛠️ Usage
1.  Open WhatsApp Web.
2.  Set a Master Password on the first run.
3.  Use the Tampermonkey menu to change settings or reset the password.
